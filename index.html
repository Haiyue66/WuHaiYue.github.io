搭建自己的第一个博客

2020-01-20
1.下载nodejs

登陆nodejs.org，下载并安装，安装的时候不停的下一步就行了
安装好后会出现nodejs和npm管理器
打开cmd，输入node -v查看node版本,npm -v查看包管理器的版本

2.利用npm安装cnpm：

输入npm install -g cnpm —registry=http://registry.npm.taobao.org
等待安装，安装完成后输入cnpm -v查看当前版本，确认下载成功

3.利用cnpm安装hexo框架：

输入cnpm install -g hexo-cli
这样hexo博客框架安装完毕

4.开始正式搭建博客

先创建一个新的文件夹blog，如果在创建的过程中出现什么问题，直接把blog删掉重来，输入cd blog，进入blog路径下，输入hexo init，表示初始化一个博客，必须在管理员模式下进行，等待一段时间，这时框架已经安装成功了，输入hexo s，表示启动博客
然后可以自己修改博客，记得hexo clean表示清理，hexo g表示生成，然后hexo s启动

5.部署到github

部署前准备：下载github，然后配置git的用户名和邮箱（具体方法自己百度），配置ssh公钥
5.1在github上创建一个库，库的名字必须和自己的用户名一致，在dos命令窗口输入：cnpm install —save hexo-deployer-git ,装完之后，修改blog当中的_config.yml文件，在文件的最底部，加入
type: git repo:(库的地址） banch: master ,修改完成后dos窗口输入hexo d，这时候电脑里面的网页相关的文件会被传送到github的库里面，输入相关网址即可访问
